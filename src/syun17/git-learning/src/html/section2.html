リモートリポジトリとローカルリポジトリ

リモートリポジトリ

インターネット上にある複数人で共有するためのリポジトリです。
共有フォルダと同じイメージです。

ローカルリポジトリ

各個人のPC内で編集するためのリポジトリです。
ダウンロードしてきたファイルと同じイメージです。

共有フォルダに例えると gitのイメージを共有フォルダに例えると次のような手順になります。 共有フォルダをダウンロードしてくる (リモートリポジトリ->ローカルリポジトリ) ダウンロードしてきたフォルダ内のファイルを編集する (ローカルリポジトリ) 編集したファイルを保存する。(ローカルリポジトリ) ファイルを選びアップロードする。(ローカルリポジトリ->リモートリポジトリ)
