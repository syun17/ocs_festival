〜Section 5〜

branchについて学ぼう

「新しい機能を追加したい」、「修正をしたいけどアプリ全体に影響するかもしれない」
こういった時にbranchを使用します

branchのイメージ

branchのメリット

1.新機能や修正を別ブランチで試せるため、メインコードが壊れない。
2.チーム全員がそれぞれのブランチで作業できる。
3.完成した機能ごとにブランチをまとめられるので、変更の流れが分かりやすい。

注意点

main-branchは基本的に編集には使わず、新たにbranchを作成してください!!
もしmain-branchで編集してプログラムが動かなくなった際に、修正が大変なことになります!!
